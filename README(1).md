
# PreviewMe signup page

A sign-up page for PreviewMe Signup 


## Tech Stack

**Client:** React, CSS


## Run Locally

Clone the project

```bash
  git clone https://github.com/mjalowe09/PreviewMe.git
```

Go to the project directory
```bash
  cd signup-form
```

Install dependencies
```bash
  npm install
```

Run the website

```bash
  npm run dev
```
